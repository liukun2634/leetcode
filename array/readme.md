# 数组总结

## 技巧总结

- 二分查找
- 哈希表
- 排序
- 优先队列
- 双指针 (判断是否有环，排序+滑动窗口)
- 动态规划
- 差分数组（特殊）
- 标记判断是否有环（增加visited/OFFSET原数组）
- 环形数组（将数组扩展一倍，使用 index % n 取对应下标)
